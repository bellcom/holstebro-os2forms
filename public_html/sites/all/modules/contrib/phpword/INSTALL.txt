 - Download PHPOffice-PHPWord library (https://github.com/PHPOffice/PHPWord), unzip and rename folder to "PHPOffice-PHPWord".
 - Put the folder in a libraries directory (example: sites/all/libraries/PHPOffice-PHPWord).
 - Install and enable Libraries API 2.x (https://drupal.org/project/libraries).
 - Install and enable X Autoload (https://drupal.org/project/xautoload).
 - Install and enable PHPOffice-Common (https://drupal.org/project/phpoffice_common).
 - Install and enable PHPOffice-PHPWord module.
